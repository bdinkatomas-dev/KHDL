# 🌲 Konfederace Horního a Dolního lesa (KHDL)
**Oficiální státní repozitář a digitální archiv**

Vítejte na půdě Konfederace Horního a Dolního lesa. Tento prostor slouží k transparentní správě mikronároda, uchovávání legislativy a monitorování stavu lesního biomu.

* **Správce:** @bdinkatomas-dev
